Stock Lot Rules

Automate lot date calculations with configurable nomenclature rules

🎯 Overview

The Stock Lot Rules module implements an automated system for calculating lot expiration dates based on the lot name nomenclature. This eliminates manual date entry and reduces human errors in inventory management.

✨ Key Features

  • Automatic Date Calculation: Extracts manufacturing dates from lot names and calculates all related dates
  • Multiple Nomenclatures: Supports AAMM-PROD_CODE and AAMMDD-PROD_CODE formats
  • Configurable Rules: Define custom rules for different product categories
  • Validation: Ensures lot names follow the configured format
  • Industry Templates: Pre-configured rules for food, pharmaceutical, and standard industries
  • Extensible: Easy to add new nomenclature types in the future

📋 Flexible Format Patterns

The module uses a flexible format pattern system similar to ir.sequence that allows you to define custom lot nomenclatures using placeholders:

Available Placeholders

  • %(yy)s: 2-digit year (e.g., 25 for 2025)
  • %(yyyy)s: 4-digit year (e.g., 2025)
  • %(mm)s: 2-digit month (e.g., 02 for February)
  • %(dd)s: 2-digit day (e.g., 15)
  • %(product_code)s: Product code placeholder

Format Examples

%(yy)s%(mm)s-%(product_code)s
Creates: 2502-ABC123 (February 2025)
%(yy)s%(mm)s%(dd)s-%(product_code)s
Creates: 250215-ABC123 (February 15, 2025)
%(yyyy)s/%(mm)s/%(dd)s_%(product_code)s
Creates: 2025/02/15_ABC123 (February 15, 2025)
%(dd)s.%(mm)s.%(yy)s.%(product_code)s
Creates: 15.02.25.ABC123 (February 15, 2025)

🚀 How It Works

  1. Configure Rules: Go to Inventory → Configuration → Lot Rules
  2. Assign to Products: Select a lot rule in the product's inventory tab
  3. Create Lots: When creating lots, dates are automatically calculated from the lot name
  4. Validation: The system validates that lot names follow the configured format

📊 Date Calculations

For each lot rule, you can configure:

  • Expiration Days: Days until the product expires
  • Use Days: Days for best consumption period
  • Removal Days: Days until product should be removed from inventory
  • Alert Days: Days to trigger alerts before expiration

🏭 Industry Templates

The module includes pre-configured rules for different industries:

  • Food Industry: 90-day expiration with early consumption warnings
  • Pharmaceutical: 3-year expiration with extended alert periods
  • Standard: 1-year expiration for general products

⚠️ Important Notes

  • Once a lot rule is assigned to a product, the date fields become read-only
  • Manufacturing dates cannot be in the future
  • Lot names must follow the exact format specified in the rule
  • The module requires products to have lot tracking enabled

🔧 Configuration

  1. Install the module
  2. Navigate to Inventory → Configuration → Lot Rules
  3. Create or modify lot rules according to your needs
  4. Assign rules to products in their inventory configuration
  5. Start creating lots with automatic date calculation

👥 Security

  • Stock Users: Can view lot rules
  • Stock Managers: Can create, modify, and delete lot rules

📈 Benefits

  • Reduces manual data entry errors
  • Ensures consistent date calculations
  • Improves inventory management efficiency
  • Supports regulatory compliance
  • Standardizes lot nomenclature across the organization
