Payroll Dual Period
===================

This module adds functionality for a dual payroll period, allowing HR administrators to define two separate date ranges
for each payroll cycle: one for incident tracking and another for official reporting.

This feature is especially useful in scenarios where payroll calculations cover a specific range (e.g., November 1 to
November 15), but data for incidents is reviewed from an earlier date range due to processing requirements
(e.g., October 28 to November 13).

**Dual Payroll Period**: Set two date ranges for payroll – one for internal tracking of incidents and another for
official documentation.
**Enhanced Reporting**: Display the additional date range in the payroll PDF report.
**Flexible Workflow**: Ideal for workflows requiring approval or adjustments prior to final payroll submission.

Usage Example
-------------

For a payroll period spanning November 1-15:

**Incident period**: October 28 to November 13 (internal tracking and validation).
**Payroll period**: November 1 to November 15 (official dates for reporting).

This module ensures a smooth payroll management experience while maintaining compliance with regional reporting standards.

Installation
============

To install this module, you need to:

- Not special pre-installation is required, just install as a regular Odoo
  module:

  - Download this module from `Vauxoo/hr-advanced
    <https://github.com/vauxoo/hr-advanced>`_
  - Add the repository folder into your odoo addons-path.
  - Go to ``Settings > Module list`` search for the current name and click in
    ``Install`` button.
